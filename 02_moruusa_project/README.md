# モルウサ判定AI

機械学習の基本概念を学ぶために作成した学習ノートです。

題材として「耳の長さ」と「体重」から、
新入り動物がモルモットかウサギかを判定します。

---

## 学習内容

### 01_dataset.ipynb
- データセット作成
- 特徴量とは何か
- モルモットとウサギの学習データ作成

### 02_knn.ipynb
- k-NN（k近傍法）
- ユークリッド距離
- 距離計算
- 散布図による可視化
- 新入り動物の分類

### 03_logistic_regression.ipynb
- ロジスティック回帰
- シグモイド関数
- 確率による分類
- predict()
- predict_proba()
- KNNとの違い
